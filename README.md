# nodeloc-Run

   | Secret 名称       | 说明                                      |
   |--------------------|-------------------------------------------|
   | `NODELOC_USER`     | 你的 NodeLoc 登录邮箱/用户名                |
   | `NODELOC_PASS`     | 你的 NodeLoc 登录密码                       |
   | `TG_BOT_TOKEN`     | Telegram 机器人 Token（找 @BotFather 申请） |
   | `TG_CHAT_ID`       | 接收通知的 Telegram 会话 ID                 |
